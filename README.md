# Bosh

Bosh is fake mosh.
It can automatically reconnect to the host with the remote GNU screen session.

## Installation

Add this line to your application's Gemfile:

    gem 'bosh'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bosh

## Usage

    bosh hostname [GNU_screen_session_name]
